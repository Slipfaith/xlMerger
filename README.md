# xlMerger

## English

xlMerger is a Qt based utility for merging translation spreadsheets, checking string length limits and splitting Excel files.

### Installation

```bash
pip install -r requirements.txt
```

### Usage

Run the application:

```bash
python main.py
```

To run tests:

```bash
pytest
```
